Chỉ cần 4 bước đơn giản để thêm game mới:

  1. � Copy file JAR vào thư mục web/games/
  2. ✏️ Chỉnh sửa file web/games/list.json
  3. 🔄 Refresh trang web
  4. 🚀 Chơi ngay! Game tự động được cài đặt

Ví dụ cấu hình:

{
  "filename": "ninja3.jar",
  "name": "Ninja School 3",
  "jadFile": null,
  "settings": {
    "phone": "Nokia",
    "width": "240",
    "height": "320",
    "sound": "on"
  }
}

💡 Tip: Hầu hết game chạy tốt với Nokia và kích thước 240x320

⌨️ Phím điều khiển trên PC

🎮 Điều hướng
↑ ↓ ← → Di chuyển (Lên/Xuống/Trái/Phải)
⏎ Enter Nút OK / Xác nhận
Esc Mở/đóng menu tùy chọn
📱 Bàn phím điện thoại
0 đến 9 Phím số 0-9
E hoặc * Phím * (sao)
R hoặc # Phím # (thăng)
F1 hoặc Q Phím mềm trái (Soft Left)
F2 hoặc W Phím mềm phải (Soft Right)
Numpad (0-9) Bàn phím số (với 123 và 789 hoán đổi)

💡 Lưu ý: Phím * và # được dùng trong nhiều game để mở menu, chuyển vũ khí, hoặc các tính năng đặc biệt.
